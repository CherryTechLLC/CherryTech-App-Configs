# CherryTech-App-Configs
The respository contains the .env and .yaml files for all the apps we use at CherryTech.

# Using this Resource
These configurations are not sufficient for internet exposed instances. The configurations in the repository are designed to be used on a CentOS Stream server with Podman for container management. We use Caddy as a Reverse Proxy, and DNS-01 through Hetzner to get Let's Encrypt web certificates.

CherryTech LLC. does not use generative AI or other models for creating production facing code or configurations. AI may be used for non-production work or research.
